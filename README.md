Netflix-Clone-Frontend-Website
 Netflix Clone

This is a frontend Netflix clone created using **HTML and CSS**.  
The project is designed to replicate the basic layout and user interface of the Netflix website.

It includes a homepage with a hero section, trending content, feature highlights, a FAQ section, and a separate sign-in page.

This project was built for learning and practice purposes to improve frontend web development skills.

Features
- Netflix-style homepage layout  
- Navigation bar with Netflix logo  
- Email input and call-to-action section  
- Trending movies section  
- Feature cards section  
- Frequently Asked Questions (FAQ) section  
- Separate sign-in page  

Technologies Used
- HTML  
- CSS  

Purpose
This project was created to practice:
- HTML page structure  
- CSS styling and layout  
- Real-world website UI design
